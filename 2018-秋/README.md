# 2018 秋

## 1. 文件的组织方式

每周每位同学分享的内容及相关材料，位于一个文件夹下，文件夹命名方式为“{第几次组会}.{序号} {主题} [{分享人}]”，如“1.1 情感分析简介 [张三]”。
